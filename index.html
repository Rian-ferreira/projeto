Projeto em Destaque

Zabbix Cloud

Este projeto tem como objetivo implementar o Zabbix, uma poderosa plataforma de monitoramento de código aberto, em um ambiente cloud distribuído, escalável e com alta disponibilidade.

A arquitetura foi desenhada para integração com Grafana, múltiplos Zabbix Proxies para coleta descentralizada de dados, monitoramento de redes, servidores, aplicações e serviços críticos. O projeto incluiu automações, alertas inteligentes, personalização de dashboards e scripts avançados para garantir a proatividade na detecção e solução de incidentes.

Toda a solução roda sobre containers, permitindo atualizações sem downtime e fácil expansão. Ferramentas utilizadas: Docker, PostgreSQL, Nginx, Zabbix API, Telegram Bot para alertas e documentação em Markdown.
